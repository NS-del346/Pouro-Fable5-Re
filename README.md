# Pourō-Fable5 Re

**App name:** Pourō  
**Description:** A quiet hand-drip coffee brew guide and timer.

## Status

Planning / Governance Foundation

This repository does not yet contain a runtime application. The PWA, brew timer, recipe engine, storage layer, and user interface are approved or planned product scope only; they are not available for use from this repository at the current stage.

## Authority

- Google Drive holds the approved planning authority until governance and domain specifications are migrated through reviewed pull requests.
- GitHub is the authority for the repository, branches, commits, pull requests, diffs, and checks.
- After the governance foundation is merged, GitHub will become the authority for code, runtime, PR state, QA evidence, releases, and Current Status.
